# BotOfTheSpecter Discord Bot - Version 5.8 (2025-11-07)

## Version Information

* Updated version from 5.7 to 5.8
* Upgraded AI module and chat handling for improved responsiveness, reliability, and streaming responses from OpenAI client

## New Features & Major Changes

### AI Module Upgrade

* Upgraded to asynchronous OpenAI client (AsyncOpenAI) with streaming support
* Implemented new instructions endpoint and short-lived instructions caching for faster prompts
* Added per-user chat history directory support with persisted chat history for context
* Implemented streaming AI response path (server-to-client streaming) with helper `get_ai_response_stream` function

### Message Handling Improvements

* Improved message splitting that preserves Markdown formatting when sending long replies
* Enhanced error handling and safer directory/logging setup with explicit ensure-directory checks
* Minor refactors for AI-related utilities and error handling

### System Enhancements

* Added comprehensive logging and connection validation
* Improved compatibility with client-side handlers for streaming responses
