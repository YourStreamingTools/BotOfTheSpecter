# Overview

BotOfTheSpecter is a comprehensive Twitch chat bot designed to enhance the streaming experience on Twitch. It offers a wide range of functionalities, from chat interaction and moderation to in-depth analytics and user management.

**Note:** Changelogs for versions prior to 5.6 have not been created. We acknowledge that documenting previous changes was overlooked, and we apologize for this oversight. Moving forward, we are committed to providing changelogs for updates.
