# Changelog - Specter Discord Bot

*Note: Versions 5.5 and 5.6 are combined in this changelog as version 5.5 was never released as a separate update.*

## Version 5.6 - Reaction Roles

- **New Feature: Reaction Roles** - Implemented the `Reaction Roles` system using the `RoleButtonView` class and `RoleButton` components. Users can now click on reaction buttons to automatically assign themselves roles in Discord servers.
- **Server Management Enhancements** - Continued development of the `ServerManagement` class to support role assignment and reaction role functionality.
- **Database Integration** - Added support for storing reaction role configurations in the database for persistent role management.

## Version 5.5 - User Auto Role and Welcome Message Configuration

- **New Feature: User Auto Role Assignment** - Implemented automatic role assignment for new members joining Discord servers using the `ServerManagement` class.
- **New Feature: Welcome Message Configuration** - Added customizable welcome messages that are posted to specified channels when new users join, integrated through the `ServerManagement` class.
- **Server Management Foundation** - Established the `ServerManagement` class as the core component for handling server-specific features and configurations.
- **Event Handling** - Enhanced event listeners in the `ServerManagement` class to detect member joins and trigger welcome messages and role assignments.

## Future Updates

- **Version 6.0 Coming Soon** - More features are on the way! Version 6.0 is well underway and will complete all server management features including:
  - **Role History** - Configure role restoration settings for members who rejoin your Discord server
  - **Message Tracking** - Message tracking for edited and deleted messages in your Discord server
  - **Role Tracking** - Role change tracking for audit purposes in your Discord server
  - **Server Role Management** - Tracking for role creation and deletion within your Discord server
  - **User Tracking** - User profile change tracking for your Discord server, Track nickname changes, Track avatar changes, Track status changes and more
