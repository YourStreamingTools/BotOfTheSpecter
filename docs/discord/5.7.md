# BotOfTheSpecter Discord Bot - Version 5.7 Release Notes

**Release Date:** October 15, 2025

## Server Rules Configuration

(Check it out in #rules )

- Dashboard configuration for server rules messages
- Post customizable rules with title, content, and embed color
- Users accept rules by clicking green "ACCEPT RULES" button
- Automatically assigns configured role when rules are accepted
- Buttons are persistent and reliable
- All user feedback is ephemeral (only they see it)

## Welcome Message Colors (NEW FEATURE)

**Brand new feature released today!**

- Color picker for welcome message embeds
- Default color: Turquoise (#00d1b2)
- Color picker only shows when embed mode is enabled
- Customize embeds to match your server branding
