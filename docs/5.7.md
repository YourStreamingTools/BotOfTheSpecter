# Changelog - Version 5.7 (2025-12-15)

## Version Information

* Updated version from 5.6 to 5.7
* Enhanced EventSub integration and moderation event handling

## New Features & Major Changes

* **Complete Twitch EventSub channel.moderate v2 Implementation**
  * Added full support for all moderation actions including timeouts, bans, warnings, role changes (mod/vip), message deletions, and chat mode toggles
  * Comprehensive logging of all moderator actions with detailed information (user, reason, moderator, timestamps)
  * Real-time event processing with proper error handling and fallbacks

* **Discord Integration for Moderation Events**
  * Moderation events are now automatically forwarded to Discord servers via websocket
  * Rich embed formatting in dedicated moderation channels with color-coded actions
  * Includes moderator name, affected user, action type, and relevant details
  * Supports shared chat moderation events from multi-channel setups

* **Enhanced Websocket Communication**
  * Improved event broadcasting system for real-time notifications
  * Better error handling and connection management for websocket clients
  * Optimized payload structure for moderation event data transmission

## Code Improvements

* Refactored EventSub message processing to handle complex moderation payloads
* Added robust error handling for unknown or malformed moderation events
* Improved logging granularity for better debugging and monitoring
* Enhanced Discord embed formatting with appropriate colors and icons for different moderation actions
